## Ubuntu

To install stuff on ubuntu

1.  Install fonts
2.  Copy `.zshrc` and `.gitconfig` and `.hyper.js` to `$HOME`

3.  Run ubuntu/install_me.sh `sh ubuntu/install_me.sh`
